# Birdazzone Full

Project for the University of Bologna Data Structure and Algorithms course
(a.y. 2020-21).

# Run

Production:

```bash
docker-compose -f docker-compose.yml -f production.yml up
```

Development:

```bash
docker-compose -f docker-compose.yml -f development.yml up
```
